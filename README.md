# Pingpong Frontend

## 技術棧

- **React** 19 - 前端框架
- **Vite** 7 - 建置工具
- **Tailwind CSS** 4 - CSS 框架
- **React Router DOM** 7 - 路由管理
- **TanStack React Query** - 資料請求管理
- **Axios** - HTTP 請求
- **Lucide React** - Icon 圖示庫
- **Sonner** - Toast 通知

## 開始使用

### 安裝依賴
```bash
npm install
```

### 啟動開發伺服器
```bash
npm run dev
```

### 建置正式版本
```bash
npm run build
```

### 預覽正式版本
```bash
npm run preview
```
